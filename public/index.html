Lập trình Python bằng phần mềm Notepadd++

Hiện nay có rất nhiều phần mềm hỗ trợ cho việc lập trình tuy nhiên với các bạn mới làm quen với lập trình thì Notepadd ++ là phần mềm mà các bạn có thể sử dụng trước khi sử dụng những phần mềm khác.

Mặc định các bạn đã có thể lập trình đc Python trên Notepadd ++ rồi nhưng khi chạy chương trình chúng ta phải mở cửa sổ cmd hoặc powershell trên windows ra để chạy. Để thuận tiện cho việc lập trình, trong bài viết này mình sẽ hướng dẫn các bạn cách mở cửa sổ console bên dưới phần code để thuận tiện cho việc chạy chương trình. Với việc mở cửa sổ console này thì các bạn cũng có thể lập trình java thuận tiện hơn trên Notepadd ++.

Đầu tiên các bạn vào menu Plugins => Plugins Admin

Sau đó chúng ta cài đặt plugin NppExec bằng cách tích vào ô vuông và click Install ở góc bên phải cửa sổ

Sau khi cài đặt xong để hiện cửa sổ console chúng ta vào Plugins -> NppExec -> Show console như hình.

Cửa sổ console đã hiện ra.

Tuy nhiên để chạy được chương trình chúng ta phải chạy câu lệnh cmd để bắt đầu sử dụng.

Sau đó chúng ta đánh tên file python đã lập trình sẵn và chạy chương trình.

Thay vì gõ lệnh cmd các bạn cũng có thể sử dụng PowerShell của windows.

Khi chạy chương trình nếu ta chỉ đánh tên file sẽ xảy ra lỗi vì thế các bạn phải thêm ./ vào trước tên file. Ở đây của mình là ./hehe.py

Responsive với mixin trong Sass

Responsive website có thể không phải là vấn đề gì lớn đối với những người làm frontend lâu năm nhưng với những người mới bắt đầu thì thực sự không phải là dễ.

Khi là việc với Sass ta có thể tạo 1 Mixin đơn giản như sau phục vụ cho việc Responsive

Ta sử dụng như sau

Việc lựa chọn các breakpoint tùy thuộc và dự án của các bạn. đây chỉ là 1 ví dụ mình đưa ra.

Tuy nhiên thì dường như nó có vẻ đơn giản khi mà ta đang sử dụng Sass. Một Css preprocessors có thể làm được rất nhiều điều. Ví dụ như ta sẽ rút ngắn những dòng lệnh trên.

Đầu tiên ta sẽ tạo 1 map dữ liệu breakpoint và screen

Ở đây chúng ta sẽ lấy giá trị breakpoint của các màn hình hiển thị và in ra media queries

Đối với desktop first hoàn toàn tương tự. Ta thêm hàm if - else để kiểm tra screen nhập vào có chính xác không

Nếu muốn mix cả desktop first và mobile first ta thêm chút xíu nữa sẽ được như này
